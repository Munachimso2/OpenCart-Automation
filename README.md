# OpenCart Automation

End-to-end test automation framework for the OpenCart ecommerce application.

## Tech Stack
- Playwright
- JavaScript
- Node.js
- Page Object Model
- GitHub Actions

## Test Coverage
- Login
- Register
- Logout (in progress)
- Search (in progress)
- Cart (in progress)
- Checkout (in progress)

## Setup
npm install
npx playwright test
